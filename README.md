# keyChat
一个基于golang的匿名聊天室程序

随意输入用户名 和 房间号，会根据房间号进入聊天室

聊天所有的记录不会被保存。

this is a anonymous chatting program,

input a username and a roomid then you can go to a chatting room which is identified by the roomid that you typed.

By beego and websocket
